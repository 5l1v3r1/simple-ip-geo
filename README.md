# simple-ip-geo
Simple IP Geolocation<br>
<pre>
Installation : 
$pkg update && upgrade
$pkg install php
$pkg install git
$git clone https://github.com/soracyberteam/simple-ip-geo/
$cd simple-ip-geo
$php ip.php

Usage :
php ip.php [IP] [Mode]
Example : php .ip.php 127.0.0.1 2
IP is 127.0.0.1
Mode is 2

Mode : 1 -> ipinfo.io
       2 -> ip-api.com
       3 -> ip-stack.com
</pre>
